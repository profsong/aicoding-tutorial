## AI 모델별 코딩 튜토리얼

### 💎 Gemini 코딩 튜토리얼

> **Google** · [`gemini.html`](gemini.html) · 7개 챕터

Gemini CLI, Antigravity IDE, Google AI Studio, Stitch를 활용한 풀스택 개발 가이드입니다. 윈도우 환경에서 무료로 Gemini 2.5 Pro를 사용하여 서비스를 개발하고 클라우드에 배포하는 전 과정을 단계별로 안내합니다.

**다루는 도구:**

`Gemini CLI` · `Antigravity` · `AI Studio` · `Stitch` · `Cloud Run`

**챕터 구성:**

| 챕터 | 주제 | 내용 |
|:---:|------|------|
| 01 | **핵심 기술 스택** | 프론트엔드/백엔드 개념, React, TypeScript, FastAPI, SQLite, npm 소개, 윈도우 개발 환경 세팅 |
| 02 | **Gemini CLI** | 설치, 인증(무료 1,000회/일), 대화형·원라인 사용법, GEMINI.md 컨텍스트 파일 |
| 03 | **Antigravity** | 에이전트 중심 IDE 설치·설정, Editor/Manager View, Plan/Fast 모드 활용 |
| 04 | **AI Studio** | 웹 기반 Vibe Coding, 멀티모달 입력(이미지→코드), API Key 발급 |
| 05 | **Stitch** | AI UI 디자인 생성, 커스터마이징, Figma/코드 내보내기, Antigravity 연동 |
| 06 | **디자인 카피 전략** | 토스·배민·카카오 등 디자인 시스템 분석법, CSS 변수 테마 구축 |
| 07 | **클라우드 배포** | 8개 옵션 비교(무료~프로덕션), Vercel + Cloud Run 실전 배포 가이드 |

👉 **[Gemini 튜토리얼 바로가기 →](gemini.html)**

## 🚀 시작하기

### 사전 준비 (윈도우)

```powershell
# Node.js LTS 설치 (npm 포함)
winget install OpenJS.NodeJS.LTS

# Python 3.12+ 설치
winget install Python.Python.3.12

# Git 설치
winget install Git.Git

# 설치 확인
node --version     # v20.x 이상
python --version   # 3.12 이상
git --version
```

### 로컬에서 튜토리얼 보기

```bash
# 리포지토리 클론
git clone https://github.com/YOUR_USERNAME/ai-coding-tutorial.git
cd ai-coding-tutorial

# 브라우저에서 열기 (VS Code Live Server 또는 직접 열기)
start index.html
```

또는 GitHub Pages로 배포된 사이트에서 바로 확인:  
🌐 **https://YOUR_USERNAME.github.io/ai-coding-tutorial/**
